Welcome

Banking Customer Prediction System is a Machine Learning base project working on a customer of a bank is fulfill the criteria to get loan from the bank. This can be used by banking authority to check customer's requirement is fulfill or not. This project also include a customer exit prediction system that can be predict by taking some customer's data that the customer will leave the bank or not. A AI voice assistant is integrated in this system for any help the bank authority to search any query in internet over voice.
